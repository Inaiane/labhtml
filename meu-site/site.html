ARTIGOS

UX Design

Publicado em: 1 de Janeiro de 2014

O livro Introdução e boas práticas em UX Design de Fabricio Teixeira, traz uma abordagem muito interessante, sobre como podemos utilizar ferramentas e metodos para imprimir o UX em um produto. Dentro dessa abordagem, percebe-se que experiência do usuário deve ser desenvolvida desde o inicio do projeto do produto, para isso,mesmo que tenha Designer de UX na equipe do projeto, é necessário que todos as pessoas envolvidas no projeto também busque desenvolver o produto pensando no UX.

Além de reuniões para discustir sobre o produto, é necessário conhecer o público potencial para esse produto, quais são os beneficios que o produto vai oferecer a essas pessoas. No dia a dia, do desenvolvimento será definido como aplicar uma boa hieraquia, como possibilitar a acessibilidade, como melhorar a usabilidade. Assim, se percebe quais técnicas e métodos são eficazes para conseguir um UX no seu produto.

Por: Inaiane Almeida
